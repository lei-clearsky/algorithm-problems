# Snake Game
A simple classic snake game

## How to view the app?
- Use Node 8.9.0 and `npm install`
- `npm run build`
- Open `/public/index.html` to view local app
- or see live [DEMO](https://snake-game-app.herokuapp.com/public/index.html) deployed on Heroku
